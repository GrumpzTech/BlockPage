How to sign up/join and make a site

Read thoroughly so you know what to do! If you need any help let us know!

  If you already know how to make a site and use GitHub/GitHub Desktop, just read the first 3 lines.

Start an issue here

If you know other ways to contact us, such as on Escargot, you can use that as well.

In the issue, ask to join, we will then invite you to collaborate on the site

If you do not get the invite, reply to us on the issue and we will try our best to help you!

Download GitHub Desktop if you want and login to the account you started the issue with.

If you have Github Desktop, then click File and then clone the repository once you accept the invite

Make a new folder in the subsites folder if using the website,

You will need to make a file along with the folder by putting the name of your site and then "/" then type a name of your placeholder file, you can delete it later.

if using Github Desktop, Make the folder in the clone of the repository under the subsites folder. MAKE SURE YOU KNOW WHERE THE CLONE IS SAVED

If you do not know how, please reply to your issue to request to join or start a new one.

Then download an HTML editor or WYSIWYG editor of your choice and make your site in the your cloned subsite folder!

Then commit your changes using GitHub Desktop and push to the repository

If you chose to not use GitHub desktop, upload your files to the folder you made in the subsites folder using the website.

Congratulations on making your new site, have fun and good luck!
